​In this article, we will connect to Interactive Brokers via the ib_insync Python module, fetch an option chain, and identify potential box spread opportunities.

Prerequisites

To follow along, you'll need:

  • An Interactive Brokers account.
  • Python 3.6 or later with the ib_insync library installed. You can install it with pip install ib_insync.
  • Trader Workstation (TWS) or IB Gateway software running on your machine, available for download from the Interactive Brokers website.

Why TWS or IB Gateway?

Interactive Brokers API needs a running session of either TWS or IB Gateway to operate. They act as a bridge between the API and Interactive Brokers' server. Both are similar, but they have some differences:

  • TWS has a graphical interface and is designed for manual traders who want to click buttons to make their trades.
    You can download it here.
  • IB Gateway is a lightweight, less resource-intensive solution that is designed for automated trading. It has a minimal user interface.You can download it here.

For this guide, you can use either TWS or IB Gateway, but I recommend using TWS in the beginning to have an overview what is happening when you are placing orders via the ib_insync Python module.

Connecting to Interactive Brokers

For our purposes we are importing the python modules pandas, numpy and we are importing everything from the ib_insync Python module, indicated by the * and create an instance of the IB class:

import pandas as pd
import numpy as np

from ib_insync import *

# util.startLoop() # uncomment this line in a jupyter notebook

ib = IB()

Next, we connect to Interactive Brokers:

ib.connect('127.0.0.1', 7497, clientId=1)

In this command, '127.0.0.1' is the IP address (localhost), 7497 is the port, and clientId is an identifier for the client. If you're using TWS, the port is usually 7496 or 7497, and if you're using IB Gateway, it's typically 4001 or 4002. Make sure you replace these values with the ones you use.

After running the connect command, if everything is set up correctly, you should be connected to your Interactive Brokers account.

In the following we will look at the most liquid options market, the S&P 500 index. With high volume and tighter spreads, the S&P 500 is a good starting point for our quest to find box spread opportunities.

[chain] = [c for c in chains if c.tradingClass=='SPX' and  c.exchange=='SMART']

Getting Our Options Universe

Now, we proceed to specify the strikes, expirations, and rights of the options we want to trade:

strikes = [strike for strike in chain.strikes if strike % 50 == 0

           and spx_price - 500 < strike < spx_price + 500]

expirations = sorted(exp for exp in chain.expirations)

rights = ['P', 'C']

contracts = [Option('SPX', expiration, strike, right, 'CBOE', tradingClass='SPX')

        for right in rights

        for expiration in expirations

        for strike in strikes]

We are interested in strikes that are a multiple of 50 and within a range of the current SPX price. We sort all the available expirations, specify the rights we're interested in (puts and calls), and then construct a list of Option objects for these strikes, expirations, and rights. The resulting contracts list contains all the potential contracts we can trade.

Moving forward, we need to qualify these contracts and fetch their respective tickers:

contracts = ib.qualifyContracts(*contracts)

ib.reqMarketDataType(1) # 1: live prices, 3: delayed prices 4: delayed end of day prices

tickers = ib.reqTickers(*contracts)

The function reqTickers returns us a snapshot of all tickers (where * is indicating multiple contracts). Then we can use a utility function of the ib_insync python module to convert this list of ticker snapshots into a pandas DataFrame.

df = util.df(tickers)[['contract','bid','bidSize','ask','askSize']]

 We can enrich this DataFrame with more information from the contract attribute of the ticker object from the tickers list. We are interested in the strike, the right, and the execution date:

df[["lastTradeDateOrContractMonth",'strike','right']] = util.df(

[ticker.contract for ticker in tickers])[['lastTradeDateOrContractMonth','strike','right']]

Screening the Options Universe

Now that we have this DataFrame, we can find potential box spread opportunities by calculating the outlay for creating a box spread with different strike prices. I have defined a function for this that is basically just computing the initial premium paid (long box) or premium received (short box) as you would do it by hand:

def calc_box_outlay(df, k1,k2, exp, long=True):

    # Mask for the condition - we want to make sure we have bid/ask for all options

    mask = (df["strike"].isin([k1,k2])) & (df["expiration"] == exp)

    has_null = df[mask].isnull().any().any()

    if has_null or len(df[mask])<4:

        box_outlay = None

        return box_outlay

    else:

        if long:

            long_call = df[(df['strike']==k1) & (df['right'] == 'C') & (df['expiration'] == exp)]['ask']

            short_put =df[(df['strike']==k1) & (df['right'] == 'P') & (df['expiration'] == exp)]['bid']

            synthetic_long_outlay = short_put.values[0] - long_call.values[0]


            long_put = df[(df['strike']==k2) & (df['right'] == 'P') & (df['expiration'] == exp)]['ask']

            short_call =df[(df['strike']==k2) & (df['right'] == 'C') & (df['expiration'] == exp)]['bid']

            synthetic_short_outlay = short_call.values[0] - long_put.values[0]

            box_outlay = synthetic_long_outlay + synthetic_short_outlay


        else:

            short_call = df[(df['strike']==k1) & (df['right'] == 'C') & (df['expiration'] == exp)]['bid']

            long_put =df[(df['strike']==k1) & (df['right'] == 'P') & (df['expiration'] == exp)]['ask']

            synthetic_long_outlay = short_call.values[0] - long_put.values[0]


            short_put = df[(df['strike']==k2) & (df['right'] == 'P') & (df['expiration'] == exp)]['bid']

            long_call =df[(df['strike']==k2) & (df['right'] == 'C') & (df['expiration'] == exp)]['ask']

            synthetic_short_outlay = short_put.values[0] - long_call.values[0]

            box_outlay = synthetic_long_outlay + synthetic_short_outlay +k2-k1

       

        return abs(box_outlay)

In a final step I am iterating over the different expiration days of the option contracts as well as over all possible box spread combinations within these. You can compare the current yield of the S&P 500 options market with the current yield of the US-Treasuries to stay in the somewhat riskless world and hunt some real arbitrage profits or simply use this money to invest with a little higher leverage in the stock market. The best part is that there are even profitable short box combinations (on a standalone basis) out there with odd strike combinations that human option traders wouldn't even think of to input in an option strategy constructor.
